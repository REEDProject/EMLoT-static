Viewing Event Record: The 1623 Shakespeare folio prints a list of the 'Principall Actors' in the plays

Abstract

The 1623 Shakespeare folio prints a list of the 'Principall Actors in all these Playes,' and so consolidates the identity of the Chamberlain's-King's Men company for posterity.
