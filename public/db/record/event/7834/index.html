Viewing Event Record: The Globe property is mortgaged to Matthew Browne, John Collett, and John Bodley

Abstract

Matthew Browne, John Collett, and John Bodley mortgage the land upon which the Globe playhouse is built from the Brend family. The mortgage names 'Richard Burbadge and William Shackspeare, gent.' as tenants.
